Dear all,

In lme4 2.0-1, the diag tag only suppresses estimation of the correlation in
m1, but not in m2:

item <- factor(rep(1:10, each = 18))
m1 <- lmer(Reaction ~ Days + diag(Days | Subject) + (1 | item), sleepstudy)
m2 <- lmer(Reaction ~ Days + (1 | item) + diag(Days | Subject), sleepstudy)

logLik(m1)  # 'log Lik.' -871.756 (df=6)
logLik(m2)  # 'log Lik.' -871.7482 (df=7)

Seemingly, depending on the order of the two random-effect terms one
gets different models. Is this intended behavior?

Hello all,

Long version below.

For a concrete application, I would like to generate tolerance and
prediction regions of straight lines obtained by mixed-effects linear
regression, with random slope and intercept, and including this
randoms variations also.

Is there any package that allows to do this, like a

md <- lm( Y ~ X, data = d )
predict( md, newdata = dp, interval = "pred" )

for a classical linear model?

** Long version **

The context is to determine limits of blank (LOB), detection (LOD) and
quantification (LOQ) of an analytical method that uses a calibration
curve, Y = a X + b where X is the concentration and Y the observed
value. First step is to find a and b from known (X, Y) values, and
then to determine unknown values X* of X using observed Y values,
hence X* = (Y - b)/a. Of course, X* != X even for training values.

The method is used with different reagents, leading to a
(potential) fixed effect on a and b, and in different centers, leading
to a (potential) random effect on a and b also. Limits (LOB, LOD, LOQ)
should be defined for each reagent, but for an average center, taking
into account between-center variability.

Fitting the model with lmer works quite well, and Gaussian assumptions
are reasonnably met.

LOB is the 0,95 quantile of the X* distribution when X equals 0, which
is one condition available of the data --- so it is the upper limit of
a 95% prediction, or eventually tolerance, unilateral interval.
Currently, I reconstruct it ? by hand ? adapting the basic formula:

IP = m + t * s * sqrt( 1 + 1/n )

with m = average, s = standard deviation, t quantile of a Student
distribution with n-1 ddl and n the sample size. Adaptation is to
define s? as the sum of residual variance and between-center variance,
and usethe Kendall-Rogers approximation of the dl.

Is there anything better than that?

LOD is defined as the lowest X for which the 0.05 quantile of X*
distribution equals the LOB. Hence, it may be obtained by finding the
intersection of the vertical line ? X = LOB ? and the boundary of the
90% prediction interval of the straight line [upper if a > 0, lower if
a < 0].  Hence my question on how to obtain the prediction region of
the straight line, for a given reagent but taking into account both
between-center and residual variability?

Eventually, would ? forgetting ? the random effect and using a
classical linear model, for which predict() works, give a reasonnable
value, or problems of correlations between values would make this
highly unreliable?

Thanks in advance for any help,
